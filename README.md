# ME Thesis Manuscript – AI-Assisted Musculoskeletal Ultrasound Analysis

This repository contains the full manuscript, code, figures, and data pipeline for the
Master of Engineering thesis:

**AI-Assisted Segmentation and Quantitative Analysis of Portable Musculoskeletal Ultrasound for Stroke Rehabilitation**

Author: Vishal Goundar  
Supervisors: Prof Alan Wang, Dr Fatemeh Esmaeili, Dr Beau Pontre  
Auckland Bioengineering Institute, University of Auckland

## Structure
- `manuscript/` – Markdown chapters of the thesis
- `figures/` – All figures used in the manuscript
- `data/` – Raw and processed ultrasound datasets
- `code/` – Python scripts for segmentation, analysis, and reporting
- `docs/` – Final PDF exports and supplementary documents
